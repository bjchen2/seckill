# seckill
慕课网，SSM整合案例，Java高并发秒杀API<br/>

**使用前，请按照自己的设置将resource->jdbc.properties文件参数修改，并下载redis**<br/>
本案例涉及知识点：Spring、SpringMVC、Mybatis、Redis简单入门和存储过程（优化高并发）、md5加密的简单用法（生成秒杀接口）<br/>
存储过程和建表sql已分别写在schema和seckill.sql文件中<br/>
